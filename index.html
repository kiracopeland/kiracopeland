It's Lovely to Meet You

My name is Kira Copeland, a Front-End web designer from Canberra. I'm currently working towards the completion of my Bachelor of Web Design and Production from the University of Canberra.

I simply love designing and working on the Web, aiming to create works that are modern, flexible, and elegant. I'm always up for a challenge, and I love figuring out the best solution for any given project.

What I Offer to You

Experience

I have been working with the Web for 7 years now, using languages such as HTML, CSS, JavaScript, AJAX and PHP just to name a few. My perferred applications of choice are Adobe Creative Suite, as well as libraies and frameworks such as React, Bootstrap, etc.

Creativity

I pride myself on being able to create designs that are rich in both visual design and functionality. I'm a firm believer that everything should look as beautifully as it performs.

Enthusiasm

I've known I wanted to be a web developer ever since I changed the background colour of a website when I was 12. Since then, I've been challenging myself to be as productive and resourceful as possible when it comes to the web, and it's technologies.

Attention to Detail

I believe attentiveness is one of the most important aspects to have as a developer and designer, as it allow us to find innovative and artistic solutions. Attention to one's self is also important to allow efficient communication and preparation.

Take a Look at my Works

Seasonal

Website Design and Development

Seasonal is a responsive website that was designed and built in my spare time to gain practical experience with the Bootstrap framework, style sheet language Sass, and the extensions Animate.css and WOW.js.

Artventure

API Web Application and UI/UX

Artventure is a responsive Web application created to educate users about the various art pieces located around Canberra, while drawing design and functional inspiration from the popular 2016 application, Pokémon Go.

Darksky

API Web Application and UI/UX

The aim of this project was to make a one-page web application that displays relevant weather-focused data in a dynamic and visually stimulating form.

ACT Cycle Safe

API Web Application and UI/UX

ACT Cycle Safe is a web application designed to help Canberra cyclists plan out their intended route safety, visually highlighted by cyclist fatality data and facility data.

Yuni

Application Design and Logo Design

The goal of this project was to create an application prototype with the intent of decreasing the overall dropout rate of first year university students, partnered with a clear visual branding that reflects the University of Canberra.
